The Pedelec SDK gives web applications a typed way to communicate with the local Pedelec bridge and the coding agents it connects to.

```ts title="Example only"
// The final API may differ. This snippet demonstrates the intended shape.
const session = await pedelec.connect();
await session.send({ prompt: 'Explain this project.' });
```
