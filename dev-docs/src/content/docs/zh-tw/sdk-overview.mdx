Pedelec SDK 為 Web App 提供型別化的方式，用來與本機 Pedelec bridge 及其連接的 coding agents 溝通。

```ts title="僅為示範"
// 最終 API 可能不同；此片段僅展示預期的使用形式。
const session = await pedelec.connect();
await session.send({ prompt: 'Explain this project.' });
```
